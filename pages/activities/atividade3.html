Lista

HTML
Linguagem de marcação usada para estruturar conteúdo na Web
  • Elementos: São as "tags" que definem o conteúdo;
  • Atributos: Definem características específicas de um elemento;
  • Tags principais:<html>, <head>, <body>, <div>, <p>;
CSS
Linguagem usada para estilizar o conteúdo do HTML
  • Seletores: Definem quais elementos serão estilizados;
  • Propriedades: Características a serem aplicadas (como cor, tamanho);
  • Valores:O valor atribuído à propriedade (como "red" para cor);
JavaScript
Linguagem de programação que adiciona interatividade às páginas web
  • Funções: Blocos de código reutilizáveis;
  • Eventos: Ações como cliques ou rolagem que disparam funções;
  • Objetos:Contêm dados e funcionalidades relacionadas;
SEO (Search Engine Optimization)
Conjunto de práticas para otmizar um site e melhorar sua posição nos motores de busca
  1. Palavras chaves: Termos que as pessoas procuram;
  2. Meta tags: Informações sobre página, como descrição e título;
  3. Links internos e externos:Melhoram a autoridade do site;
