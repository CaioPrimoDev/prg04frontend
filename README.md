# prg04frontend
Repositório criado para registrar o frontend do projeto final do semestre, da disciplina de Desenvolvimento Web, ministrada pelo professor Jonatas Bastos, no IFBA Campus Irecê

## Funcionalidades

- [ ] Botão "Meus Ingressos"
  - [ ] Abre um modal com todos os ingressos válidos do usuário

- [ ] Termos de Uso
  - [ ] Tela/modal/aviso com os Termos de Uso do Ingresso Eletrônico

- [ ] Tela de Compra

  - [ ] Ao clicar em "Comprar ingressos" no modal "Meus Ingressos", abrir no primeiro filme disponível

- [ ] Aviso de Classificação 18 anos
  - [ ] Modal de aviso para filmes proibidos para menores

- [ ] Tela/Modal "Total a pagar"
